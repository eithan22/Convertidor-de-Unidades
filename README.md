# Convertidor de Unidades 🔄

A **web-based unit converter** built with **HTML5, CSS3, and JavaScript**. Allows users to convert values between different units of measurement through a clean, interactive interface.

---

## 📋 Description

This project implements a unit conversion tool in the browser. Users enter a numeric value, select the source and target units, and see the converted result instantly. Built as a practice exercise in JavaScript logic, form handling, and DOM interaction.

---

## ✨ Features

- 🔄 **Unit Conversion** — Convert between multiple units of measurement
- ⚡ **Instant Results** — Real-time conversion on input
- 🎨 **Clean Interface** — Styled with CSS for a modern look
- 📱 **Responsive** — Works on desktop and mobile browsers
